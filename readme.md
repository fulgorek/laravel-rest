# Simple Laravel - JWT API

Requirements

- PHP 5.5.x
- Composer

## Installation

Install all dependencies

```composer install```

Run database migrations

```php artisan migrate```

Seed the database

```php artisan db:seed```


## Notes:
Default credentials

Username: ```admin@site.com```

Password: ```password```
